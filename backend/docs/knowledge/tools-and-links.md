# Community-built tools

| Tool | Link | What it does |
|---|---|---|
| **Z7Lab/spark-cli** | https://github.com/Z7Lab/spark-cli/blob/main/reports/reference/RESULTS.md | CLI to fully control a DGX Spark so agents can drive it — HF model downloads, playbooks, benchmarks. Public benchmark results. |
| **Z7Lab/LLM-probe** | https://github.com/Z7Lab/LLM-probe/blob/main/TESTING.md | Tests tool-calling reliability across models/providers. 8 categories (structured tool_calls, enum constraints, 3→11 tool selection, reasoning leakage, system-prompt adherence, degenerate output, latency). 3 runs, 80% pass threshold → "safe model" list. |
| **jedbjorn/md-converter** | https://github.com/jedbjorn/md-converter · live: https://md-converter.designs-os.com/?t=dossier | Themed markdown → polished shareable HTML, no backend (SvelteKit/Cloudflare). 9 themes, gzip-in-URL sharing, GitHub deep-links, Mermaid, callouts, stat cards.
